# Warehouse Inventory Management System

## 📌 Description
This is a simple Warehouse Inventory Management System developed using Java.
It uses Java Collections Framework (HashMap) to store and manage inventory items.

## 🚀 Features
- Add items
- Remove items
- Update quantity
- Search by ID or name
- Display all items

## 🛠 Technologies Used
- Java
- OOP Concepts
- Java Collections Framework

## ▶️ How to Run
1. Compile:
   javac *.java

2. Run:
   java Warehouse
